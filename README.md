# Draw me a Sheep
This repository contains the pictures, data, script and figures for the work of Andres, Fueyo & Kiefer on testing if people drew tails for sheep. 

The data was collected on Saturday 13th of May, 2023 in Jardin du Luxembourg, Paris, France.
